설치 안내

최초 1회만 필요합니다.

Microsoft Store로 설치하면 이 과정이 없습니다. 아직 서명 인증이 없는 .exe/.dmg 직접 다운로드에만 해당합니다 — 아래대로 한 번만 통과하면 됩니다.

공통“일반적으로 다운로드되지 않는 파일” 경고가 뜨면

브라우저 다운로드 목록에서 계속(유지)을 선택하면 됩니다.

WINDOWSSmartScreen “PC를 보호했습니다” 화면이 뜨면

추가 정보 → 실행을 누르면 설치가 진행됩니다.

MACOS“열 수 없음 / 확인되지 않은 개발자” 안내가 뜨면

FlowDesk를 한 번 실행하고 안내 창을 닫은 뒤, 시스템 설정 → 개인정보 보호 및 보안 아래쪽의 “그래도 열기”를 누르고 다시 실행합니다. macOS 14 이하는 응용 프로그램 폴더에서 우클릭 → 열기로도 됩니다.

공통이후 업데이트는 어떻게 하나요

Microsoft Store 설치본은 스토어가 자동으로 업데이트합니다. 직접 설치본은 새 버전이 나오면 위젯이 알려주고, 놓쳤다면 설정 → 정보 → 업데이트 확인으로 언제든 직접 설치할 수 있습니다.
