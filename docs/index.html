MACOS“열 수 없음 / 확인되지 않은 개발자” 안내가 뜨면

FlowDesk를 한 번 실행하고 안내 창을 닫은 뒤, 시스템 설정 → 개인정보 보호 및 보안 아래쪽의 “그래도 열기”를 누르고 다시 실행합니다. macOS 14 이하는 응용 프로그램 폴더에서 우클릭 → 열기로도 됩니다.
