# FlowDesk — 바탕화면 생각 정리 위젯

바탕화면에 상주하는 위젯에서 생각을 **Staging(임시) → Origin(보관) → Calendar(일정)** 으로
드래그해 정리하는 앱. 어디서든 `Cmd/Ctrl+Shift+Space` 퀵 입력으로 생각을 캡처하세요.
